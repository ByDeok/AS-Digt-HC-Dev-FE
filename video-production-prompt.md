# 골든 웰니스 10초 소개 영상 제작 프롬프트

## 영상 제작 프롬프트 (AI 영상 생성 도구용)

```
10초 길이의 시니어 웰니스 및 가족 건강 관리 플랫폼 소개 영상을 제작해주세요.

**타겟 오디언스**: 60세 이상 시니어와 그 가족 구성원

**영상 구성 (10초 타임라인)**:

0-2초: 오프닝
- 따뜻하고 신뢰감 있는 골든/웜톤 색감
- "골든 웰니스" 로고나 브랜드명이 부드럽게 등장
- 배경: 건강하고 활기찬 시니어의 실루엣 또는 가족이 함께하는 따뜻한 장면

2-5초: 핵심 기능 소개
- 건강 리포트 아이콘/그래픽이 나타나며 "건강 리포트" 텍스트 표시
- AI 코칭 아이콘과 함께 "AI 맞춤 코칭" 텍스트
- 가족 보드 아이콘과 함께 "가족과 함께" 텍스트
- 각 요소가 순차적으로 페이드인/아웃되며 전환

5-8초: 사용자 경험 강조
- 시니어가 스마트폰으로 앱을 사용하는 모습 (실루엣 또는 그래픽)
- 가족 구성원들이 함께 건강 정보를 확인하는 따뜻한 장면
- 디바이스 연동 아이콘 (Samsung Health 등)이 부드럽게 표시

8-10초: 클로징
- "골든 웰니스와 함께 건강한 삶을" 메시지 표시
- 로고가 다시 등장하며 마무리
- CTA (Call to Action) 없이 브랜드 인상만 남김

**스타일 가이드**:
- 색상: 골든/웜톤 (황금색, 따뜻한 베이지, 부드러운 오렌지)
- 톤: 따뜻하고 신뢰감 있는, 전문적이지만 친근한
- 애니메이션: 부드럽고 자연스러운 전환 (페이드, 슬라이드)
- 타이포그래피: 읽기 쉬운 큰 글씨, 명확한 한글 폰트
- 음악: 부드럽고 편안한 배경음악 (옵션)

**중요 메시지**:
1. 시니어 친화적 (간단하고 명확)
2. 가족 중심 (함께하는 건강 관리)
3. AI 기반 개인화 (맞춤형 서비스)
4. 신뢰감과 따뜻함
```
