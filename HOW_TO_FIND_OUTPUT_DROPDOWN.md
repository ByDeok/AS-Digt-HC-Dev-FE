# Output 패널 드롭다운 찾기 가이드

## Output 패널 열기

### 1단계: Output 패널 열기
1. **`Ctrl+Shift+U`** 키를 누르세요
   - 또는 메뉴: View → Output

### 2단계: 드롭다운 찾기

Output 패널이 열리면:
- **화면 하단**에 패널이 나타납니다
- 패널의 **우측 상단**에 작은 **드롭다운 메뉴**가 있습니다
- 텍스트로 "OUTPUT" 또는 "Language Support for Java" 같은 라벨이 보입니다

### 위치 설명:
```
┌─────────────────────────────────────────┐
│  [OUTPUT]  ← 이 라벨 옆에 작은 ▼ 버튼   │
├─────────────────────────────────────────┤
│                                          │
│  여기에 로그 메시지가 표시됩니다...      │
│                                          │
└─────────────────────────────────────────┘
        ↑
     드롭다운 버튼 (▼)
```

---

## 드롭다운에서 선택할 항목

드롭다운을 클릭하면 다음 항목들이 보입니다:

1. **Language Support for Java** ← 이걸 선택!
2. Test Runner for Java
3. Gradle for Java
4. Extension Host
5. 기타...

## 단계별 안내

1. **`Ctrl+Shift+U`** 누르기
2. 화면 **하단**에 Output 패널이 열림
3. **"OUTPUT"** 라벨 옆의 **▼ 버튼** 클릭
4. 목록에서 **"Language Support for Java"** 선택
5. 아래에 나타나는 메시지 확인

---

## 찾기 어려우면

### 방법 1: View 메뉴 사용
1. 상단 메뉴바: **View** 클릭
2. **Output** 선택
3. 패널 하단에서 드롭다운 찾기

### 방법 2: Command Palette로 열기
1. `Ctrl+Shift+P`
2. `View: Toggle Output` 입력
3. Output 패널이 열림

---

드롭다운을 찾으셨나요? 찾으셨으면 "Language Support for Java"를 선택해서 오류 메시지를 확인해주세요!
